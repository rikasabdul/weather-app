Weather App

A simple weather app that lets you search for the current weather of any city in the world. The app is built with HTML, CSS and JavaScript.
Features

    Search for any city to get its current weather
    Display the current temperature, humidity and wind speed
    Display a descriptive icon for the current weather conditions

How to use

    Clone the repository or download the files
    Open the index.html file in a web browser
    Type the name of the city you want to search in the input field
    Hit the Search button to get the current weather of that city

Technologies used

    HTML
    CSS
    JavaScript
    OpenWeatherMap API

Note

The app uses OpenWeatherMap API to get the weather data. The API has a limited number of requests per day, so the app might not work if the limit is exceeded. In that case, you can sign up for a free account on OpenWeatherMap to get a higher number of requests per day.
